Detailed Description

Abstract interface for energy calculation. For batches of only one to a couple calculations, use the Immediate batch type. For batches with a large number of calculation, the Queued batch type may provide better performance if the EnergyCalculator supports parallelism across hardware threads.

virtual BatchImmediate LfnIc::EnergyCalculator::BatchOpenImmediate ( const BatchParams &  params  )  [pure virtual]

BatchOpenImmediate BatchOpenQueued

Opens an energy calculation batch based on the given width and height, and left-top coordinate of the block against which the rest of the blocks are computed against (block A). If aMasked is true, then only the pixels intersecting with the known regions of block A are factored into each result.

Only one batch may be open at a time. Asserts that no batch is open.

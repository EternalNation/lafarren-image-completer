Detailed Description

Image completion settings; provided by the client code that is using the image completer.

Member Data Documentation

Belief LfnIc::Settings::confidenceBeliefThreshold

The threshold applied to a node's labels' beliefs, above which labels are considered potential candidates for completion. The fewer labels with beliefs above this threshold, the more confident the node is about finding a good candidate, thus the node is given higher priority during execution.

bool LfnIc::Settings::debugLowResolutionPasses

If enabled, each low resolution pass will generate a separate output image.

int LfnIc::Settings::latticeGapX

The gap between nodes in the Markov Random Field lattice. Both values must be >= LATTICE_GAP_MIN.

const int LfnIc::Settings::LOW_RESOLUTION_PASSES_AUTO = -1 [static]

Constants

int LfnIc::Settings::lowResolutionPassesMax

The number of low-resolution passes to take, or LOW_RESOLUTION_PASSES_AUTO to automatically determine when to stop. The more low resolution passes, the faster the completion will be at the expense of solution accuracy.

int LfnIc::Settings::patchWidth

The dimensions of the patches used to complete the unknown region of the image. Both dimensions must be >= PATCH_SIDE_MIN, and (patchWidth * patchHeight) must be <= PATCH_PIXELS_MAX.

int LfnIc::Settings::postPruneLabelsMin

The min and max numbers of labels a node should after have pruning. Both of these values must be >= POST_PRUNE_LABEL_MIN and <= POST_PRUNE_LABEL_MAX.

Belief LfnIc::Settings::pruneBeliefThreshold

The threshold applied to a node's labels' beliefs, below which labels are pruned away from being candidates for that node.

Energy LfnIc::Settings::pruneEnergySimilarThreshold

The threshold applied to the energy between two labels, one of which has been spared pruning, below which the other label is considered too similiar to the first, and is thus pruned.

Detailed Description

Encapsulates the state and execution of the Priority-BP (believe propagation) algorithm over the Markov Random Field.

Member Function Documentation

void LfnIc::PriorityBpRunner::Run (  ) 

Runs Priority-BP to completion, without populating any patches. This can be used during lower resolution solves in order to prune the label set from each node.

void LfnIc::PriorityBpRunner::RunAndGetPatches ( std::vector< Patch > &  outPatches  ) 

Executes the Priority-BP to completion, and populates the outPatches object based on the solution. The patches are sorted by a recommended order in which they should be applied to the output image.

Detailed Description

A mask that stores its data in multiple levels of detail, and is therefore able to perform efficient tests over an entire region.

Member Function Documentation

int LfnIc::MaskLod::GetHighestLod (  )  const [inline]

Returns the level of detail range that this mask supports. 0 is always the highest level of detail, where each region is 1x1 pixel and thus contains no indeterminates. Lower levels of detail are identified by successive > 0 values. A region's max size at any given lod is (2^lod)x(2^lod).

Detailed Description

Partially implements Image under the assumption that the image is const. Used internally to access read-only image data, including scaled down data, during completion.

Member Function Documentation

LfnIc::Image::Pixel * LfnIc::ImageConst::GetData (  )  [virtual]

Non-const and const access to the image's Rgb buffer. The result points to the upperleft-most pixel and the buffer is stored in row-major order, where the start of each row is separated by GetStride() bytes.
